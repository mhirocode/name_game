# name_game

## 概要
**name_game** は、プレイヤーが自分の名前をサイトに書き込み、  
リアルタイムで順位を競うシンプルなオンラインゲームです。

## ルール
- プレイヤーは自分の名前をサイトに書き込みます。  
- ランキングの **一位になった人が勝ち** です。  
- ただし、最終的な勝者は  
  **「トータルで一位の時間が最も長かった人」** になります。  
- 一位の座をできるだけ長くキープすることが目的です。

## 特徴
- ランキングがリアルタイムで更新されます。  
- シンプルながらも、駆け引きと競争が楽しめます。  
- 誰でもすぐに参加できる軽量な設計。

## ライセンス
このプロジェクトは自由に利用・改変できます。
